# Quick Lorem Ipsum Generator

A terminal-based tool to generate placeholder text instantly. Perfect for developers and designers who need content fillers without leaving the command line.

## Features

*   **Customizable Length**: Generate as many paragraphs as you need.
*   **Standard Start**: Includes the classic "Lorem ipsum dolor sit amet..." by default.
*   **Random Variation**: Ensures standard Latin-looking word distribution.

## Usage

```bash
python run_generator.py [options]
```

### Options

*   `--paragraphs`, `-p`: Number of paragraphs (default: 1).
*   `--no-start`: Randomize the beginning instead of using the standard "Lorem ipsum..." phrase.

### Examples

**1. Generate one paragraph**
```bash
python run_generator.py
```

**2. Generate 5 paragraphs**
```bash
python run_generator.py -p 5
```

**3. Generate random text without the standard start**
```bash
python run_generator.py --no-start
```

*   Python 3.x
